PROJECT TITLE:	Adivinar

PURPOSE OF PROJECT:
Programa en Java en el que el usuario debe adivinar un número que ha pensado el ordenador. El número estará entre 0 y 100 y el usuario tiene 10 intentos para adivinarlo.

VERSION or DATE: 1.0 (Oct 2016)


HOW TO START THIS PROJECT:
Proyecto BlueJ. Abre el fichero package.bluej, compílalo y ejecuta el programa principal de la clase Adivinar  o de la clase AdivinarAtras (variante con bucle hacia atrás en la que indica cuantos intentos quedan)

AUTHORS: IIP (@mrebollo)

USER INSTRUCTIONS:
Cuando se solicite, introduce un número entero positivo. El ordenador indicará si el número que ha pensado es mayor o menor. Tienes 10 intentos para adivinarlo.
